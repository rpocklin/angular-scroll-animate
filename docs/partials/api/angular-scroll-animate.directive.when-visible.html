when-visible
directive in module angular-scroll-animate

Description

Allows method hooks into the detection of when an element is scrolled into or out of view.

Usage

as attribute
<ANY when-visible
     when-not-visible="{string}"
     [when-not-visible="{string}"]
     [delay-percent="{number}"]
     [delay-percent-viewport="{number}"]
     [delay-pixels="{number}"]
     [bind-scroll-to="{string}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
when-not-visiblestring

function to execute when element is scrolled into viewport

when-not-visible
(optional)
string

function to execute when element is scrolled out of viewport

delayPercent
(optional)
number

percentage (of px) to delay animate when visible transition.

delayPercentViewport
(optional)
number

percentage (of viewport height) to delay animate when visible transition.

delayPixels
(optional)
number

to delay animate when visible transition.

bindScrollTo
(optional)
string

override default scroll event binding to another parent container.
